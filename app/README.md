## Commands
Install libraries
```bash
pip install numpy
pip install matplotlib
pip install PyQt5
```

Add images resources
```bash
pyrcc5 -o header_rc.py images/header.qrc
```

Converting the .ui file to a .py file using pyuic5
```bash
pyuic5 dialog.ui -o dialog.py
```
